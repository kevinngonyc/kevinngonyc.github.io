Clustering Of Political Journalists On Twitter
Currently using: Spectral Clustering

Learn more We aimed to observe the relationships between various political journalists through unsupervised machine learning.

Each node represents a political journalist.

All nodes belonging in the same cluster share the same color (the actual colors have no significance).

The distance between two nodes represents their "similarity", using shared followers as the primary metric.
